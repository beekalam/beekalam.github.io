1.9 BIOS Versus Bootloader

The BIOS is a complex set of system-configuration software routines that have knowledge of the low-level details of the hardware architecture.

Here are some of the more important tasks your bootloader performs on power-up: • Initializes critical hardware components, such as the SDRAM controller, I/O controllers, and graphics controllers. • Initializes system memory in preparation for passing control to the operating system. • Allocates system resources such as memory and interrupt circuits to peripheral controllers, as necessary.
